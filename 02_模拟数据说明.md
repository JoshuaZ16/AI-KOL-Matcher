# 模拟数据说明

## 数据类型

CSV 表格文件

---

## 建议数据量

**80-100 个达人数据**

覆盖 4 个平台（小红书、抖音、B站、微博），多个内容领域，不同粉丝量级和价格区间。

---

## 字段说明

| 字段 | 含义 | 示例 |
|------|------|------|
| kol_id | 达人唯一 ID | KOL_001 |
| kol_name | 达人名称 | 校园成长Ada |
| platform | 所属平台 | 小红书/抖音/B站/微博 |
| followers | 粉丝数 | 50000 |
| field | 内容领域 | 校园/职场/美妆/科技/美食/旅游/健身/母婴 |
| price | 合作报价（元） | 1200 |
| avg_likes | 平均点赞数 | 1800 |
| avg_comments | 平均评论数 | 120 |
| engagement_rate | 互动率（%） | 3.8 |
| conversion_rate | 历史转化率（%） | 3.5 |
| audience | 受众画像 | 大学生、应届生、18-25岁女性 |
| cooperation_count | 历史合作次数 | 15 |
| risk_note | 风险备注 | 广告比例较高/数据真实/无风险/粉丝质量存疑 |

---

---

## 数据用途

1. **达人库检索**：作为 AI 筛选和匹配的数据源
2. **匹配度计算**：根据受众画像、内容领域与品牌需求进行匹配度评分
3. **性价比计算**：根据报价、粉丝数、互动率、转化率计算性价比得分
4. **风险评估**：根据风险备注识别潜在风险（广告比例高、数据异常等）
5. **推荐理由生成**：基于达人数据生成具体的推荐理由
6. **ROI 预估**：根据历史转化率和报价预估投资回报比
