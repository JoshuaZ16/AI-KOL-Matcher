# 输入设计

## 输入方式

**表单输入**

用户通过填写表单的方式输入投放需求，系统根据表单内容进行达人匹配。

后续可增加自然语言输入，AI 只负责把模糊描述转换成结构化字段，再交给规则筛选和评分链路处理。

---

## 必填字段

| 字段 | 说明 | 示例 |
|------|------|------|
| 目标受众 | 本次投放的目标人群 | 大学生、职场新人、宝妈、科技爱好者 |
| 内容领域 | 希望达人的内容方向 | 校园、职场、美妆、科技、美食 |
| 预算范围 | 单个达人的预算区间（元） | 1000-3000 |
| 投放平台 | 希望投放的平台（可多选） | 小红书/抖音/B站/微博 |

---

## 可选字段

| 字段 | 说明 | 示例 |
|------|------|------|
| 粉丝数要求 | 期望的粉丝量级范围 | 5万-20万 |
| 互动率要求 | 最低互动率要求（%） | ≥3% |
| 历史转化率要求 | 最低历史转化率要求（%） | ≥3% |
| 风险偏好 | 对风险的容忍程度 | 保守/平衡/激进 |
| 合作次数偏好 | 优先选择新达人还是老达人 | 优先新达人/优先老达人/无偏好 |

---

## 用户指令样例

**样例 1：**
```
请为我们的校园活动推荐小红书达人，目标受众是大学生和应届生，预算每个达人 1000-2000 元，需要互动率高、数据真实的达人。
```

**样例 2：**
```
我们要推广一款职场技能课程，目标受众是职场新人（22-28岁），预算 2000-5000 元，平台选择抖音和B站，希望达人有过教育类合作经验，转化率不低于 3%。
```

**样例 3：**
```
为美妆新品推荐达人，目标受众是 18-25 岁女性学生党，预算 1500-3500 元，平台小红书，要求粉丝数 5-15 万，互动率 ≥3.5%，风险偏好保守。
```

**后续自然语言理解样例：**
```
想找校园感强、真实、不太商业、适合大学生的达人
```

可解析为：

| 结构化字段 | 解析结果 |
|------------|----------|
| 目标人群 | 大学生 |
| 内容风格 | 真实、日常、素人感 |
| 风险偏好 | 保守 |
| 内容领域 | 校园/生活/vlog |

---

## 输入限制

1. **必填字段完整性**：目标受众、内容领域、预算范围、投放平台必须填写
2. **预算范围合理性**：预算下限 ≥ 500 元，预算上限 ≤ 50000 元，且下限 < 上限
3. **平台选择**：至少选择 1 个投放平台
4. **文本长度**：目标受众和内容领域描述建议 10-100 字
